# douban_crawler
input the id of a user, crawl all his/her photos in albums
## dependencies
pipenv is used to control the dependencies.

1. install pipenv.
```
pip install pipenv
```
2. install dependencies
```
pipenv install
```
3. enter virtual envirment
```
pipenv shell
```
## usage
```
python AlbumCrawler [user_id]
```
